# House Price Prediction

## Objective
To predict house prices using machine learning regression models.

## Dataset
House price dataset from Kaggle.

## Steps Performed
- Data Cleaning
- Handling Missing Values
- Feature Engineering
- Log Transformation
- Encoding & Scaling

## Models Used
- Linear Regression
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- RMSE
- MAE

## Result
Best model selected based on lowest RMSE.

## How to Run
1. Open the notebook
2. Run all cells
3. Use the prediction section

## Files
- house_price.ipynb
- house_model.pkl
